# GDPD

Algorithms to retrieve land surface phenology in global dryland from MODIS NBAR EVI2 data.

The global dryland phenology dataset (GDPD) is available at https://doi.org/10.6084/m9.figshare.27160602.v2.

Paper has been publised in Scientific Data:
Dong, Y., Zhou, Y., Zhang, L., Tian, F., Xie, Q., Chen, Y., Ruan, L., Zhang, B., 2025. An Enhanced Phenology Dataset for Global Drylands from 2001 to 2019. Sci Data 12, 1–14. https://doi.org/10.1038/s41597-025-05519-2
